#define TIRION_VERSION   "0.1"

The version of the tirion client The version is also used in the communication with the agent and dictates the whole communication protocol.

typedef struct TirionStruct Tirion

Struct for a Tirion object

typedef struct TirionPrivateStruct TirionPrivate

Struct for private data of a Tirion object Tirion uses the pimpl idiom (opaque pointer) to hide the internal state from the user. This allows many independent objects without interference.
